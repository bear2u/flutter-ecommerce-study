### 목표

`Flutter ` 를 이용해서 쇼핑몰 샘플 프로젝트를 완성하는게 목표입니다. 

### 예상 시간

핵심기능을 구성하는데 약 2달 걸릴거라 예상됩니다. 

### 스터디 시간

매주 수요일 오후 8시 ~ 10시 또는 10시 30분

### 준비물

1. Flutter SDK
2. Android Studio 또는 VSCODE
3. Node
4. Docker
4. 노트북
5. Postman

### 공부할 내용

- Flutter Basic Widget
- Flutter Design Pattern
- Flutter + Server 연동
- Dart 기본 문법
- Bloc 패턴
- RxDart
- Stream, Future (비동기)
